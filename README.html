项目描述

飞机客舱产品事业部是北京飞机维修工程有限公司北京基地的部门之一，主要负责公务机的维修与改装工作。该部门的的生产管理模式为生产现场经理做出工作安排并列出人力需求，由生产计划室按照本部门的人力状态按月发布人力计划。在目前的生产管理模式中，存在无法实时控制人员状态的问题，使得对项目的进度产生影响。为此飞机客舱产品事业部就该问题申请了生产人力计划控制系统开发的科技项目。通过开发计算机应用程序保证公务机维修与改装工作的顺利开展。本次实习将协助该项目的产品设计与参与项目的软件编码工作。

项目内容

具体任务

  1. 按照部门的实际生产流程，分析应用程序的开发需求并根据开发需求确定使用的计算机程序开发技术。
  2. 利用计算机编程语言，对以下四个核心功能进行编码开发工作：
    1. 人员考勤记录
    2. 生产人力计划统计方法和跨部门数据采集
    3. 人员与生产计划匹配方法
    4. 人员计划与生产计划可视化

工作计划

工作计划共分为五个部分，根据实习的起止日期与现阶段的已经完成工作确定以下的时间安排：

时间 计划
2019-07-02至2019-07-27 采集项目需求，确定开发语言
2019-07-28至2019-08-31 实现人员生产计划匹配方法和人员计划与生产计划可视两大功能
2019-09-01至2019-10-15 实现人员考勤记录及生产人力计划统计方法和跨部门数据采集两大功能
2019-10-16至2019-11-10 优化程序性能

预期目标

项目最终要展示三项成果：

  1. 可分组统计并自动汇总的工时统计报表
  2. 自动关联的生产人力统计表格
  3. 按不同属性进行统计的生产计划图标
